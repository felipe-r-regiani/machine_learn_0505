# machine_learn_0505

Este repositório é focado na aplicação prática e análise de **Árvores de Decisão**, explorando critérios de partilha de dados e a interpretação de fronteiras de decisão em problemas de classificação.

## 📂 Conteúdo

* **dataset_ex1.csv**: Dados relativos a funcionários (idade, experiência, treinamentos) para prever a probabilidade de promoção.
* **dataset_ex2.csv**: Dados financeiros e demográficos (renda, idade, score) focados na aprovação de crédito bancário.
* **exercicio1.ipynb**: Notebook focado na análise de promoção de funcionários. O projeto abrange:
    - Identificação de padrões de experiência e idade.
    - Discussão sobre profundidade da árvore e generalização do modelo.
* **exercicio2.ipynb**: Notebook focado na análise de risco de crédito. O projeto abrange:
    - Determinação dos critérios mais relevantes (renda vs. score).
    - Visualização de fronteiras de decisão em múltiplas dimensões.

## 🛠️ Tecnologias e Bibliotecas

As ferramentas centrais utilizadas neste projeto são:

* **Python 3**: Linguagem base.
* **NumPy**: Processamento de matrizes e cálculos matemáticos.
* **Matplotlib**: Geração de gráficos e visualização das zonas de decisão.
* **CSV**: Leitura e tratamento dos datasets de treino.
* **Jupyter Notebook**: Ambiente para execução e documentação das análises.

## 🚀 Como começar

1.  Clone este repositório:
    ```bash
    git clone https://github.com/felipe-r-regiani/machine_learn_0505.git
    ```
2.  Instale as bibliotecas necessárias:
    ```bash
    pip install numpy matplotlib
    ```
3.  Execute os notebooks:
    - Abra o `exercicio1.ipynb` ou `exercicio2.ipynb`.
    - Analise as respostas teóricas incluídas ao final de cada notebook para compreender os insights extraídos de cada modelo.
